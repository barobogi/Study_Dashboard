# 만복이 News 기능 요구사항

**최초 협의**: 2026-07-03 (compact 전)  
**보완 협의**: 2026-07-04  
**저장 목적**: compact 후에도 협의 내용 유지

---

## 핵심 철학
"단순히 5개 뽑아서 올리는 게 아니라, 만복이도 실제로 읽고 기억에 반영하는 도구"

---

## 기사 목록 뷰 (만복이 News 탭)
- 매일 만복이가 직접 GeekNews 읽고 5개 선별 (자동 선별 ❌, 만복이 직접 ✅)
- 태그: [트렌드] [학습] [특허후보]
- 텔레그램 자동 전송은 n8n 완전 안정화 이후 고려 (현재는 미적용)

## 기사 상세 뷰 (제목 클릭 시)
- 원본 링크
- 만복이 요약
- 만복이가 선정한 이유 (우리 세계관 연결 포인트, 특허 연결성 등)
- 바로보기 comments 기능 (수정 가능)
- 코멘트에 대한 feedback → 다음날 아티클에 추가 반영

## n8n 자동화 범위 (현재 합의)
- GeekNews 크롤링 → 기사 목록 파싱 (자동 ✅)
- Study Dashboard 게시판 등록 (자동 ✅)
- 만복이 요약/선정이유 작성 (만복이 세션에서 직접 ✅)
- 텔레그램 전송 (나중에 고려)
